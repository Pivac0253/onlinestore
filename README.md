Описание проекта
Онлайн-магазин по продаже SUP (Stand Up Paddle) досок. Проект разработан на Django с использованием Python.

🛠 Технологии
Backend: Django 4.x

Frontend: HTML, CSS, JavaScript 

База данных: SQLite 

Дополнительно:

Корзина покупок (cart/)

Система заказов (orders/)

Магазин товаров (shop/)

onlinestore/

├── cart/               # Корзина покупок

├── orders/             # Система заказов

├── shop/               # Каталог товаров

├── media/              # Медиафайлы 

├── manage.py           # Основной скрипт управления

└── db.sqlite3          # База данных
