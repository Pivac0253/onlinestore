Описание проекта
Онлайн-магазин по продаже SUP (Stand Up Paddle) досок. Проект разработан на Django с использованием Python.

🛠 Технологии
Backend: Django 4.x

Frontend: HTML, CSS, JavaScript 

База данных: SQLite 

Дополнительно:

### Добавлен экспорт заказов в CSV
**Новый функционал в административной панели:**
- Добавлена возможность экспорта выбранных заказов в CSV-файл
- Доступно через действие "Export to CSV" в списке заказов
- Экспортируются все основные поля заказа
- Даты автоматически форматируются в удобочитаемый вид

**Как использовать:**
1. Зайдите в раздел заказов в админ-панели (`/admin/orders/order/`)
2. Отметьте нужные заказы чекбоксами
3. В выпадающем меню "Действие" выберите "Export to CSV"
4. Нажмите кнопку "Go"
5. Файл `order.csv` будет автоматически загружен


## Добавление системы купонов

Была реализована система скидочных купонов со следующими возможностями:

### Функционал
- Создание купонов через админ-панель
- Применение купонов к заказу
- Учет скидки при расчете итоговой суммы
- Ограничение по времени действия купонов

### Изменения в проекте

1. **Новое приложение `coupons`**:
   - Модель `Coupon` для хранения данных о купонах
   - Форма `CouponApplyForm` для ввода кода купона
   - Представление `coupon_apply` для обработки купонов

2. **Интеграция с корзиной**:
   - Добавлены методы в класс `Cart` для работы со скидками:
     - `coupon()` - возвращает текущий примененный купон
     - `get_discount()` - рассчитывает сумму скидки
     - `get_total_price_after_discount()` - итоговая сумма со скидкой

3. **Изменения в заказах**:
   - Добавлены поля в модель `Order`:
     - `coupon` (ForeignKey) - ссылка на примененный купон
     - `discount` (IntegerField) - размер скидки в процентах
   - Обновлен метод `get_total_cost()` для учета скидки

4. **Шаблоны**:
   - Добавлена форма для ввода купона в корзину (`cart/detail.html`)
   - Отображение скидки в процессе оформления заказа (`orders/order/create.html`)

### Настройка купонов через админку
1. Перейдите в раздел "Coupons" в админ-панели
2. Создайте новый купон, указав:
   - Код купона
   - Даты действия (valid_from и valid_to)
   - Размер скидки (0-100%)
   - Активность (active)

### Использование
1. Добавьте товары в корзину
2. Введите код купона в форме в корзине
3. Скидка автоматически применится к итоговой сумме
4. При оформлении заказа купон будет сохранен вместе с заказом
